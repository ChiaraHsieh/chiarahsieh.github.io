中央山脈極致挑戰自行車攜帶清單

2017-08-29
自行車
自行車, 裝備清單

分享我自己列的攜帶清單，如有缺漏歡迎各位留言補齊，謝謝！

自行車相關

  1. 水壺二罐(大會補給點還蠻多的，到補給點喝飽了再上，裝一瓶就夠了，除非你很會流汗，就多喝點水)
  2. 車錶
  3. 前後車燈(第二天會經過很暗的山洞)
  4. 攜車袋(這次要搭大會車下山, 需準備)
  5. 屁股救星(後擋泥板)

人身部品

  1. 安全帽
  2. 小帽/頭巾
  3. 太陽眼鏡/隱形眼鏡
  4. 車衣2~3件(快乾的車衣晚上能洗好吸水晾乾隔天再穿)
  5. 打底背心
  6. 手套
  7. 袖套
  8. 心跳帶
  9. 車褲2~3件
  10. 車襪3雙
  11. 車鞋1~2雙(如果下雨隔天不想穿濕鞋子上路)
  12. 防磨擦膏

隨車工具修胎

  1. 內胎數條(我會準備3條, 身上帶一條)
  2. 挖胎棒(我都只帶一支)
  3. 打氣筒/CO2氣瓶氣嘴(帶一瓶備用)
  4. 隨身摺疊工具
  5. 打鏈器

備用工具(不用隨身帶)

  1. 外胎兩條
  2. 鏈條油
  3. 剪刀
  4. 剎車皮(第二天長下坡可備用)
  5. 擦拭布(下雨清泥沙)

車衣口袋放的

  1. 三天份補給, 能量膠、能量棒、能量飲、羊羹鹽錠等(我會抓每一小時吃一包，慢慢騎，帶6~8包)
  2. 防水袋裝手機
  3. 零錢(中途雜貨店買飲料)
  4. 健保卡
  5. 面紙/濕紙巾
  6. 防風防水外套(山上溫差大+下雨會很冷, 捲起來用夾鏈袋包起)
  7. 防曬乳/防曬噴霧
  8. 數位相機/記憶卡

下車要用的

衣物

  1. 平底鞋/拖鞋
  2. 休閒衣褲2~3套
  3. 貼身衣物4套(前一天住宿~最後一天盥洗)
  4. 睡衣睡褲
  5. 薄外套

電器

  1. 手機充電線/插座
  2. usb延伸多插
  3. 車燈車錶充電線
  4. 行動電源
  5. 數位相機充電線

恢復用

  1. 壓縮腿套
  2. 按摩滾筒
  3. 痠痛藥膏
  4. 曬後修復乳液
  5. 眼罩/耳塞

其他

  1. 個人藥品
  2. 急救包
  3. 個人證件
  4. 現金、信用卡
  5. 雨傘/雨衣
  6. 牙膏牙刷毛巾
  7. 沐浴乳洗髮精洗衣精
  8. 衣架
  9. 跳繩(車友秘技，可晾衣及熱身，騎不動也可綁別人)
